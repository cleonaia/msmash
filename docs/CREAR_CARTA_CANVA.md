# 🎨 CREAR CARTA EN CANVA - Guía Paso a Paso

**Tiempo:** 45 minutos
**Dificultad:** ⭐⭐ (Básica)
**Resultado:** Carta profesional lista para Instagram y para imprimir

---

## 📋 ANTES DE EMPEZAR

**Abre esto en otra pestaña para copiar colores:**
```
Fondo morado graffiti: #3A1F4A
Panel negro carbón: #0F0F12
Texto crema: #F5F0E6
Acento turquesa neón: #23C7C9
Acento queso/naranja: #F59A23
Amarillo brillo: #FFD34D
```

---

## PASO 1️⃣: CREAR EL DOCUMENTO

1. **Abre Canva** → www.canva.com
2. **Clic en "Crear diseño personalizado"** (botón rojo)
3. **Dimensiones:** Escribe `1080x1350`
4. **Presiona Enter**
5. ✅ Ya tienes el canvas listo

---

## PASO 2️⃣: FONDO MORADO GRAFFITI

1. **Panel izquierdo** → Click en el cuadrado de color (bajo "Fondo")
2. **Color personalizado** → Escribe `#3A1F4A` (morado graffiti)
3. **Aplica** (click verde)
4. ✅ Ahora tu fondo es morado profesional

---

## PASO 3️⃣: AGREGAR LOGO

1. **Arriba en "Uploads"** → Click en carpeta
2. **Sube tu logo M SMASH** (o usa un placeholder)
3. Una vez subido → **Click en la imagen** → Se agrega al canvas
4. **Redimensiona:** arrastra las esquinas hasta hacer 120×120px (pequeñito, centrado)
5. ✅ Logo agregado

---

## PASO 4️⃣: TÍTULO "CARTA"

1. **Panel izquierdo** → Click "Texto"
2. **Escribe:** `CARTA`
3. **Tipografía:** Busca "Bangers" (si no está, usa "Fredoka One" u otra display)
4. **Tamaño:** 48px (arrastra para hacer grande)
5. **Color:** `#F5F0E6` (crema)
6. **Posición:** Centrado, bajo el logo
7. **Sombra:** Click en "Efectos" → "Sombra" → configura negra, offset 4px abajo
8. ✅ Título listo

---

## PASO 5️⃣: HANDLE (@msmashburguer)

1. **Nuevo elemento de texto**
2. **Escribe:** `@msmashburguer`
3. **Tipografía:** Inter (o similar sans-serif)
4. **Tamaño:** 16px
5. **Color:** `#F59A23` (naranja)
6. **Centrado, bajo el título "CARTA"**
7. ✅ Handle agregado

---

## PASO 6️⃣: PRIMERA BARRA DE CATEGORÍA ("BURGERS")

### Crear el rectángulo
1. **Panel izquierdo** → "Elementos" → "Formas"
2. **Elige:** Rectángulo redondeado
3. **Tamaño:** Ancho ~280px, alto 50px
4. **Color:** `#23C7C9` (turquesa neón)
5. **Borde:** 2px color `#000000` (negro)
6. **Sombra:** 0 4px 8px negro (busca en "Efectos")

### Agregar texto encima
1. **Nuevo elemento de texto**
2. **Escribe:** `BURGERS`
3. **Tipografía:** Bangers, 32px
4. **Color:** `#0F0F12` (negro carbón)
5. **Centro el texto encima del rectángulo**
6. ✅ Barra lista

---

## PASO 7️⃣: PRIMERA TARJETA DE PRODUCTO (The Crispy)

### Crear el rectángulo de tarjeta
1. **Elemento** → **Forma** → **Rectángulo redondeado**
2. **Tamaño:** 280×280px (para grid 2 columnas)
3. **Color:** `#0F0F12` (negro carbón)
4. **Borde:** 2px sólido `#23C7C9` (turquesa)
5. **Radio de esquinas:** 12px
6. **Sombra:** 0 8px 16px negro (70% opacidad)

### Agregar nombre del producto
1. **Texto nuevo dentro de la tarjeta**
2. **Escribe:** `THE CRISPY`
3. **Tipografía:** Bebas Neue (o similar), bold
4. **Tamaño:** 24px
5. **Color:** `#F5F0E6` (crema)
6. **Posición:** Arriba de la tarjeta, padding 20px

### Agregar precio en cápsula
1. **Forma → Rectángulo redondeado**
2. **Tamaño:** 120×40px
3. **Color:** `#F59A23` (naranja queso)
4. **Radio:** 20px (bien redondeado)
5. **Texto dentro:** `11,00€`
6. **Tipografía:** Bebas Neue, 20px, bold
7. **Color texto:** `#0F0F12` (negro)
8. **Sombra:** 0 4px 8px naranja (30% opacidad)
9. **Posición:** Derecha de la tarjeta, alineado con nombre

### Agregar descripción
1. **Texto nuevo**
2. **Escribe:** `Pan brioche, smash burger con queso gouda, bacon crujiente, nuestras salsas caseras, cebolla caramelizada`
3. **Tipografía:** Inter, 13px
4. **Color:** `#F5F0E6` (crema)
5. **Opacidad:** 85% (para dar contraste)
6. **Líneas máximas:** 2-3
7. **Posición:** Centro de la tarjeta

### Agregar chips de alérgenos
1. **Forma → Rectángulo redondeado**
2. **Tamaño pequeño:** 70×28px
3. **Color:** `#F5F0E6` (crema)
4. **Borde:** 1.5px sólido `#23C7C9` (turquesa)
5. **Radio:** 16px
6. **Texto:** `Gluten` (Inter, 12px, color `#0F0F12`)
7. **Duplica 3 veces** (una para cada alérgeno: Gluten, Lácteos, Huevos)
8. **Posición:** Abajo de la tarjeta, espaciadas

✅ **Primera tarjeta lista!**

---

## PASO 8️⃣: DUPLICAR TARJETAS

1. **Selecciona toda la tarjeta** (18 elementos aprox)
2. **Ctrl+C (Copiar)** → **Ctrl+V (Pegar)** 
3. **Mueve a la derecha** (grid 2 columnas)
4. **Edita solo:** nombre, precio, descripción, alérgenos (actualiza el texto de cada una)
5. **Repite para:**
   - The M Smash (13,00€)
   - The Basic (6,50€)
   - Super Crispy Chicken (11,90€)
6. ✅ Todas las burgers listas

---

## PASO 9️⃣: SEGUNDA BARRA ("ENTRANTES")

1. **Copia la barra "BURGERS"** (Ctrl+C → Ctrl+V)
2. **Edita solo el texto:** Cambia a `ENTRANTES`
3. **Posicion:** Bajo las 4 tarjetas de burgers
4. ✅ Barra lista

---

## PASO 1️⃣0️⃣: TARJETAS DE ENTRANTES

1. **Duplica una tarjeta de burger** (Ctrl+C → Ctrl+V)
2. **Redimensiona a:** 280×240px (más compacta, sin imagen)
3. **Actualiza:**
   - **Tequeños** - 5,00€ - "4 dedos de queso..." - [Gluten, Lácteos]
   - **Fries M** - 7,00€ - "Patatas fritas..." - [Gluten, Lácteos]
   - **Crispy Chicken** - 6,00€ - "6 tiras de pollo..." - [Gluten, Huevos]
4. **Grid:** 2×2 o 3×1 según espacio
5. ✅ Entrantes listas

---

## PASO 1️⃣1️⃣: AGREGAR SALPICADURAS GRAFFITI (OPCIONAL)

1. **Panel izquierdo** → "Elementos"
2. **Busca:** "spray", "salpicadura" o "graffiti"
3. **Elige:** formas irregulares de color
4. **Color:** `#23C7C9` (turquesa)
5. **Opacidad:** 30%
6. **Tamaño:** 30-50px
7. **Posición:** esquinas superiores, bordes laterales
8. ✅ Efecto graffiti agregado (sutil)

## PASO 1️⃣4️⃣: EXPORTAR

### Para Instagram:
1. **Botón azul "Descargar"** (arriba derecha)
2. **Tipo:** PNG
3. **Calidad:** Máxima
4. **Nombre:** `carta-msmash-instagram.png`

### Para PDF (imprenta):
1. **Cambiar tamaño a A4** (para una segunda versión)
2. **Crear nuevo documento:** 1920×2640px (A4 a 300dpi)
3. **MISMO DISEÑO** pero con 2 columnas, márgenes 15mm
4. **Descargar como PDF**
5. **Nombre:** `carta-msmash-impresa.pdf`

## 🔧 TIPS AVANZADOS

### Si quieres añadir fotos:
1. **Sube foto en "Uploads"**
2. **Agrégala al canvas**
3. **Redimensiona a 260×160px**
4. **Posición:** Arriba de cada tarjeta
5. **Radio de esquinas:** 8px

### Si quieres QR (para pedir online):
1. **Generador QR gratis:** www.qr-code-generator.com
2. **Apunta a:** tu web o WhatsApp
3. **Descarga PNG**
4. **Sube en Canva y posiciona en footer** (40×40px)

### Si quieres animaciones (para stories):
1. **Canva Pro** → Click "Animaciones"
2. **Elige:** Zoom, Slide, Fade
3. **Duración:** 0.5s - 1s
4. **Exporta como MP4** para stories
